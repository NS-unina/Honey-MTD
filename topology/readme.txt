Ordine di configurazione:

1. Eseguire lo script create_net.sh
2. Eseguire lo script setup.sh (controlla impostazioni interfaccia di rete per connessione ad internet delle macchine collegate all ovs)
3. Nella cartella /vagrant/ubuntu creare un vagrant environment (non ti dimenticare)
4. Nella cartella /vagrant/ubuntu eseguire "vagrant up"
5. Nella cartella /docker/docker-build esegurie "docker compose up"
6. Nella cartella /docker eseguire lo script setup_container.sh
